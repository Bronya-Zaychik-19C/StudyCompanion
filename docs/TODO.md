## 📝 已移除的功能

以下功能已从原计划中移除，以保持项目聚焦：

| 功能 | 移除原因 | 后续计划 |
|------|----------|----------|
| RAGFlow 知识库 | 部署复杂，非核心需求 | 暂不实现 |
| Ollama 本地模型 | GPU 门槛高 | 暂不实现 |
| Live2D 形象 | 开发成本高，非核心 | 后期可选 |
| GPT-SoVITS 音色 | 部署复杂，非核心 | 后期可选 |
| 错题本 OCR | 功能过重 | 简化为文本录入 |
| 思维导图 | 非核心 | 暂不实现 |
